# Create React App 시작하기

이 프로젝트는 [Create React App](https://github.com/facebook/create-react-app)으로 시작되었습니다.

## 사용 가능한 스크립트

프로젝트 디렉토리에서 다음을 실행할 수 있습니다:

### `npm start`

개발 모드로 앱을 실행합니다.\nOpen [http://localhost:3000](http://localhost:3000)을 브라우저에서 엽니다.

변경 사항을 저장하면 페이지가 다시 로드됩니다.\n콘솔에서 린트 오류를 볼 수도 있습니다.

### `npm run build`

프로덕션용 앱을 `build` 폴더에 빌드합니다.\n프로덕션 모드에서 React를 올바르게 번들하고 최상의 성능을 위해 빌드를 최적화합니다.

빌드는 압축되며 파일 이름에는 해시가 포함됩니다.\n앱을 배포할 준비가 되었습니다!

자세한 내용은 [배포](https://facebook.github.io/create-react-app/docs/deployment) 섹션을 참조하십시오.
